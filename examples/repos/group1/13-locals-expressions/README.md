# 13 · Locals & Expressions

Showcases advanced `locals` usage: `for` expressions, `flatten`, `merge`,
`zipmap`, `setproduct`, and `regexall`.  No cloud provider; output goes to
local JSON files for inspection.

## Concepts covered
- Complex `for` expressions and transformations
- `flatten`, `merge`, `zipmap`, `setproduct`
- `regexall` for string processing
- Building computed data sets locally

## Usage
```bash
terraform init
terraform apply
cat /tmp/tf-locals/report.json
```
