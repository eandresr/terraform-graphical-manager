Open-source local Terraform UI

Terraform Graphical Manager is an open-source Python web application that provides a Terraform Cloud-like graphical dashboard for managing Terraform workspaces stored locally on disk.

No cloud account required. No authentication. No internet connection needed. It discovers workspaces automatically, detects providers and backends, executes real Terraform commands, streams logs in real-time, visualizes plan diffs, inspects state, detects drift, renders dependency graphs, and stores execution history locally or in cloud storage of your choice.

Cookie Policy

Last updated: March 2026

Terraform Graphical Manager is a locally-hosted web application. It runs exclusively on your own machine or private network and does not connect to any external service unless explicitly directed to do so (e.g. cloud storage backends you configure yourself).

Cookies used

This application uses a single, strictly necessary session cookie (session) provided by the Flask web framework. It is used solely to support flash messages (one-time notifications after form submissions such as saving settings).

  • No tracking cookies are used.
  • No analytics or advertising cookies are used.
  • No third-party cookies are set.
  • No data from this application is transmitted to external servers.

Local storage

The application may use the browser's localStorage to preserve UI state (e.g. open tabs, panel preferences) across page loads. This data never leaves your browser and is not sent to any server.

Privacy

All data processed by this application — including your Terraform workspace files, execution logs, plan outputs, and state files — remains entirely within your local environment. Nothing is shared with third parties.
